# Ma Configuration de webpack-boilerplate
Ce dépot me permet de commencer rapidement mes projets

## Démarrage rapide sur un nouveau projet

```bash
git clone https://github.com/sopraos/webpack-boilerplate.git
cd webpack-boilerplate

make install # Installe les dépendances et lance le serveur de dev
make start # Lance le serveur de dev
make build # Construit le projet
```
